Для запуска тестов вам необходимо заполнить два файла конфигурации
1. ./src/main/resources/logopass.properties
Поле  login  - вписать свой логин на сайт Отус
Поле password - вписать свой пароль на сайт Отус

2. ./src/main/resources/personalData.properties
name - вписать свое имя (кириллицей)
LatinName - вписать свое имя (латиницей)
surname - вписать свою фамилию (кириллицей)
LatinSurname - вписать свою фамилию (латиницей)
blogName вписать свое имя для блога на сайте
birthDate - вписать дату рождения
Country - указать наименование страны из списка, доступного на сайте Отус
City - указать наименование одного из городов, принадлежащих выбранной стране и доступных для выбора на сайте Отус
englishLevel - указать уровень владения английским согласно списку на сайте Отус
relocationAbility - указать готовность к релокации (Да/Нет)
workScheduleFullDay - указать признак режима работы полного дня (Да/Нет)
workScheduleFlexible - указать признак режима гибкого графика работы (Да/Нет)
workScheduleRemote - указать признак режима удаленной работы (Да/Нет)
contactFacebook - указать контактную информацию в соцсети Facebook
contactVK - указать контактную информацию в соцсети VK

Файл предзаполнен данными в неперсонализированных полях. Перснальные данные следует заполнить самостоятельно.
При выборе допустимых режимов работы (полный день/гибкий график/удаленно) следует учитывать, что хотя бы одно поле обязательно должно быть = Да, иначе тест не будет работать.
